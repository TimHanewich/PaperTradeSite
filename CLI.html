Installing the CLI

The link above will download the CLI in a zip folder. After unzipping the folder, place the contained folder (called "PaperTradeCli") in a tucked away safe place on your computer. The installation process will save a reference to the folder, so once you install it, you cannot move the folder.

After placing the folder in a safe place, run the "pt.exe" program as an administrator (right click, run as administrator).

Running the program as an administrator will install a reference to the program for your command line. You are now ready to use the Paper Trade CLI! Open up your command line and use the 'pt' command to interface with PaperTrade.
